Gradient generator

A web application that enables the creation of a wide variety of gradients, providing users with the ability to obtain the associated CSS code.

Responsibilities

Conceptualization of application's features and functionalities, User Interface Design, Front-end Implementation, Autonomous Project Development

URL

https://isaure-lohest.com/Gradient-generator-project/Gradient-generator/index.html

This project is designed with Material Design principles, focusing on clean interfaces, bold colors, and user-friendly interactions for a seamless experience. This project offers a visually appealing, user-centric environment, prioritizing simplicity and functionality.

The application allows for both color picking and manual input of color codes.

The user can choose the type of gradient (linear or radial) and adjust various parameters such as tilt, radius, background repetition, size, shuffle color picking, etc.
